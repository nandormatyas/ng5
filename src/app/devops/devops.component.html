The Development Operations

The Development Operations is responsible to provide automated testing, automated deployment and monitoring for ongoing projects and processes.

In case of deploying front-end applications Jenkins plays the main role. After setting a webhook on the choosen repository and branch, a well configured Jenkins server will clone the repository, run the desired tests and if they pass build the app and deploy it to Amazon Web Services S3 bucket which will host a static website.
Used testing tools: CircleCI, Tape, Jest, Karma&Jasmine


Back-end deployment takes an other kind of route. After Jenkins runs the tests it creates a dockerized version of the app and sends a dockerrun file(manual) to Amazon Web Services so it can deploy the app from DockerHub to an EB or EC2 instance.
Used testing tools: CircleCI, Supertest


After configuring routes, ports, credentials, Amazon Web Services will host our whole application and after connecting a domain to it, it will be ready to be open for users.
For more information check out the documentation written by our project team at Greenfox Academy.
